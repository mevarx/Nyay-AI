# NyayAI

NyayAI is an AI-powered bias detection suite designed to analyze tabular datasets (like hiring, lending, or admissions data) for hidden fairness issues. It uses Gemini AI (via Google AI Studio) alongside Python's Fairlearn library to profile sensitive columns, calculate disparate impact ratios, and generate localized, easy-to-understand audit narratives.

## 🚀 Features

- **Automated Bias Detection:** Upload a CSV to instantly flag potential demographic disparities.
- **AI-Powered Explanations:** Non-technical, plain-English narratives explaining statistical biases, generated by Gemini 1.5.
- **Data Pre-processing Fixes:** Recommends and simulates mathematical interventions to improve dataset fairness upstream.
- **Interactive "Magic" UI:** Includes high-end React-style interactive hover buttons and seamless glassmorphism components.
- **Secure & Stateless:** AI narratives are generated server-side. No API keys are exposed to the client-side. Processed datasets and insights are strictly transient and cached session-by-session.

## 🛠️ Tech Stack

**Frontend (Client)**
- Vanilla HTML5 / CSS3 / JavaScript
- Futuristic Glassmorphism UI (Dark Mode)
- Chart.js for data visualization

**Backend (API Server)**
- Python 3.10+
- Flask & Flask-CORS
- Pandas, Scikit-Learn, Fairlearn (for statistical bias metrics)
- Google Generative AI SDK (`gemini-1.5-flash`)

## 💻 Local Setup Instructions

### 1. Backend Setup

1. Open a terminal and navigate to the backend directory:
   ```bash
   cd nyayai-backend
   ```
2. Create and activate a Python virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # Windows
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up your environment variables:
   - Create a file named `.env` in the `nyayai-backend` folder.
   - Add your Google AI Studio API key:
     ```env
     GOOGLE_API_KEY=your_real_gemini_api_key_here
     FLASK_ENV=development
     FLASK_SECRET_KEY=nyayai_dev_secret
     ```
5. Start the Flask server:
   ```bash
   python app.py
   ```
   *The backend will run on `http://127.0.0.1:5000`.*

### 2. Frontend Setup

1. Open a second terminal window and navigate to the frontend directory:
   ```bash
   cd nyayai-frontend
   ```
2. Serve the static files using Python's built-in HTTP server:
   ```bash
   python -m http.server 8000
   ```
3. **Local Testing**: 
   - Open your browser and navigate to `http://localhost:8000` (or the port specified by your server).
   - Ensure the backend is running on port 5000 for full audit functionality.

## 🧪 How to Test

1. From the Landing Page or Dashboard, click **New Audit**.
2. Drag and drop the `PRDs/sample_bias_test.csv` file into the upload zone.
3. The system will auto-detect "gender", "caste", and "religion" as sensitive features.
4. Set the outcome column to `loan_status` and click **Run Audit**.
5. Wait for the comprehensive report to generate!

## ☁️ Deployment Strategy

For the hackathon or production presentation, we recommend the following separation of concerns:

- **Frontend Deployment:** Host the `nyayai-frontend` folder on **Netlify**, **Vercel**, or **Firebase Hosting**. Update `js/config.js` to point `API_BASE_URL` to your production backend URL.
- **Backend Deployment:** Host the `nyayai-backend` folder on **Railway.app** or **Render**. Ensure you configure the Environment Variables (`GOOGLE_API_KEY`, etc.) in their respective platform dashboards.
